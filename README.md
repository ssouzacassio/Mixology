# Mixology

Sistema de gestão (caixa + estoque) da Mixology Drinkeria.

- `frontend/` — Next.js (TypeScript + Tailwind)
- `backend/` — API em Go (Gin + GORM + Postgres)

Veja o README dentro de cada pasta para instruções de configuração.

## Ordem para rodar localmente

1. Suba o backend primeiro: siga `backend/README.md`
2. Depois o frontend: siga `frontend/README.md`
