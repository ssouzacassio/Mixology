# Mixology

Sistema de gestão (caixa + estoque) da Mixology Drinkeria.

- `mix_front/` — Next.js (TypeScript + Tailwind)
- `mix_back/` — API em Go (Gin + GORM + Postgres)

Veja o README dentro de cada pasta para instruções de setup.

## Ordem para rodar localmente

1. Suba o backend primeiro: siga `mix_back/README.md`
2. Depois o front: siga `mix_front/README.md`
